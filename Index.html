Atomic Structure and the Periodic Table

Atomic Number

  • It represents the number of protons in an atom's nucleus.
  • Every element has a unique atomic number (e.g., Hydrogen is 1, Helium is 2).
  • The atomic number determines the element's identity and chemical properties.
  • It equals the number of electrons in a neutral atom.
  • Scientists use the symbol 'Z' to represent atomic number.

Mass Number

  • Mass number is the total number of protons and neutrons in an atom.
  • Represented by the symbol 'A'.
  • Mass number = Number of protons + Number of neutrons.
